INSTRUCTIONS:

1. Click on a hidden block to reveal a colored disc.

2. Strategically reveal discs by clicking on hidden blocks to create a line of four, horizontally, vertically or diagonally.

3. Aim to achieve this with two different colored discs to win.

4. Customize the grid size before starting the game.

5. Restart the game at any time if needed.

6. Have fun and aim for victory!
